# 项目当前状态

## 已实现

- **Staggered 相场断裂求解器**：在原准静态交替迭代基准上完成最小的黏性 BDF1 扩展。
  - `eta = 0` 保持原准静态行为；`eta > 0` 在 AT2 损伤方程中组装 `eta / dt` 质量项，并在右端使用上一已接受物理时间步的 `d_old`；
  - 支持独立的 `enforce_irreversibility` 开关，可运行准静态+不可逆、黏性+无不可逆、黏性+不可逆三种模式；
  - 支持 `dt`、`total_time`、非均匀 `time_points` 与 `load_history(t)`；其中 `dt` 与 `time_points` 互斥；
  - 已接受历史场与时间步试算历史场分离，仅在收敛后提交；时间步不收敛时回滚位移、损伤与历史场并停止；
  - `return_diagnostics = true` 时输出物理时间、损伤范围、单步损伤增量、黏性/累计黏性耗散、交错迭代次数和收敛状态；
  - `scripts/run_staggered.jl` 已公开 `ETA`、`N_STEPS`、`DT`、`TOTAL_TIME`、`TIME_POINTS` 和 `load_history(t)` 配置，并将诊断写入 JLD2。
- **Miehe–RLM-PE–BDF1 求解器**：实现二维平面应变、Miehe 应变谱分解、AT2 裂纹能、准静态位移平衡与黏性相场真实时间演化。
  - 外载支持位移、体力、牵引的独立分段线性时间历史；
  - 每个物理时间步只推进一次 RLM 更新，不再进行固定载荷伪时间松弛；
  - 保存并传递 RLM 辅助变量 `q` 和预测非线性能量 `P`；
  - 组装四个仿射分支并用稳定二次求根恢复物理场；
  - 输出反力、损伤、物理/代理能量、外功、黏性耗散、BDF1 数值耗散和功—能残差。

## `run_rlm_bdf1.jl` 最新结果与时间步分析

- 当前脚本参数为 `final_time = 1.3`、`ramp_time = 1.0`、`dt = 1.0e-5`、`alpha = 1000`、`viscosity = 0.001`，即 `130000` 个真实时间步；损伤方程中的 BDF1 系数为 `viscosity / dt = 100`。
- `data/plots/rlm_bdf1.png` 展示出：反力在约 `t = 0.902`、位移约 `0.721 mm` 处达到约 `348 N` 峰值；随后进入软化，加载结束时约 `258 N`，在位移保持阶段继续松弛到 `t = 1.3` 时约 `158 N`。外功在加载结束后保持约 `161.07`，累计黏性耗散约 `13.28`，累计 BDF1 数值耗散约 `0.565`。
- 这条曲线不是无条件的物理参考解：诊断中 `max_d` 曾达到 `1.0156`（`t ≈ 0.979`），`min_d` 达到 `-2.36e-4`，且 `healing` 最大约 `6.94e-4`；保持阶段末仍有 `max_d ≈ 1.0115`、`min_d ≈ -4.21e-5`。这直接反映出当前 RLM 尚无 `0 ≤ d ≤ 1` 和不可逆约束，图中的峰后松弛包含愈合与越界效应。
- 对同一配置进行无输出时间步对照后，求解器都可以完成，但响应没有时间步收敛：

  | `dt` | 真实步数 | 末时刻 `|reaction|` [N] | 历史最大 `d` | 最大 `healing` |
  | ---: | ---: | ---: | ---: | ---: |
  | `1e-3` | 1,300 | 402.27 | 1.7455 | 8.5093 |
  | `5e-4` | 2,600 | 400.36 | 1.7897 | 6.0142 |
  | `1e-4` | 13,000 | 385.92 | 1.1468 | 0.9009 |
  | `1e-5` | 130,000 | 157.84 | 1.0156 | 0.000694 |

- 原因是当前 RLM-BDF1 每个物理时间步只做一次更新，没有起裂附近的伪时间内迭代或自适应重试；增大 `dt` 同时增大载荷跳跃，并把损伤演化中的 `viscosity / dt` 降低为 `10`、`1` 等量级，因而会跨过快速起裂/软化区。故 `dt = 1e-5` 是目前能解析出图示响应的细步长，而不是已经证明收敛的“正确步长”；从 `1e-4` 到 `1e-5` 仍发生定性变化，当前结果应标记为细步长原型结果。

## 当前限制

- RLM 相场尚未实现区间约束 `0 ≤ d ≤ 1` 或不可逆约束 `d^{n+1} ≥ d^n`。
- 起裂附近可能出现 `d>1`、`d<0` 与愈合，导致非物理的峰后反力恢复；因此当前 RLM 适合作为真实时间 RLM 原型和能量/率效应研究，不应直接作为完整粘性断裂预测结果。
- 当前 `dt` 敏感性很强；尚未实现起裂阶段的自适应时间步、按损伤增量/愈合量拒绝并重试，也没有完成从 `1e-4` 到 `1e-5` 以下的时间步收敛研究。
- 尚未实现 RLM 参数 `α` 系统标定、CN 二阶格式、重启、重新网格化或位移惯性。
- 后续优先方向：先用峰值反力、峰后反力、`max_d`/`healing` 和能量项做时间步收敛扫描；然后实现与 RLM 能量律兼容的损伤障碍/不可逆约束、起裂阶段自适应时间步，以及 `q` 与代理—物理能量差诊断。
- Staggered 黏性模型尚未加入损伤区间约束 `0 ≤ d ≤ 1`、自适应时间步或 BDF2/CN；当前失败策略为完整回滚后停止，不会自动缩小时间步重试。
